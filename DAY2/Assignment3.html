Various Data types in Javascript

  1. Number - It includes integer and floating numbers.
  2. BigInt - It is used to represent integers of arbitrary length and is represented by appending ‘n’ to the end of the integer.
  3. String - It is used for strings. A string may have zero or more characters.
  4. Boolean - It is logical data type having only 2 values: True, False.
  5. Null - The value is null.
  6. Undefined - The value is not assigned.
  7. Object - It is used for more complex data structures.
  8. symbol - It is used for unique identifiers.
